synth_witch 2 days ago
Hey everyone! I've been trying to get some slow, evolving ambient textures out of my Model D reissue. The filter self-oscillation is great but I'm struggling to get long attack/decay envelopes to feel smooth enough. Anyone have go-to patch ideas or settings that work well for ambient?
analog_ben 1 day ago
Try setting OSC 3 to a very low frequency (below audible) and using it as a slow LFO to modulate the filter cutoff. Run the output through some reverb and you get that classic slowly-breathing texture.
modular_maria 18 hours ago
+1 on the OSC3 trick. Also experiment with feedback via the external audio input — feeding the output back creates unpredictable harmonic drift that works great for ambient.
