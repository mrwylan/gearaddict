# Use Case: Start Discussion Thread

## Overview

**Use Case ID:** UC-015
**Use Case Name:** Start Discussion Thread
**Primary Actor:** Home Studio Owner
**Goal:** Open a new discussion thread on an equipment page to share experience or ask a question
**Status:** Approved
**Requirements:** FR-009

## Preconditions

- The user is logged in.
- The equipment item exists in the catalog.

## Main Success Scenario

1. User navigates to an equipment page and selects "Start a Discussion".
2. System displays the new thread form with fields for a title and an opening post body.
3. User enters a thread title and writes the opening post.
4. User submits the form.
5. System validates the submission (title and body are non-empty and within length limits).
6. System creates the discussion thread attributed to the user.
7. System navigates the user to the newly created thread, displaying the opening post.

## Alternative Flows

### A1: Missing Title or Body

**Trigger:** User submits the form with the title or body empty (step 4)
**Flow:**

1. System highlights the missing field(s) and displays validation messages.
2. User fills in the missing content.
3. Use case continues at step 4.

### A2: Title or Body Exceeds Maximum Length

**Trigger:** A field exceeds the allowed character limit (step 5)
**Flow:**

1. System highlights the offending field and displays the character limit.
2. User shortens the content.
3. Use case continues at step 4.

### A3: User Not Logged In

**Trigger:** An unauthenticated visitor attempts to start a thread
**Flow:**

1. System redirects the visitor to the log-in page.
2. After successful authentication, use case resumes at step 2.

## Postconditions

### Success Postconditions

- A new discussion thread exists on the equipment page attributed to the user.
- The thread is immediately visible to all visitors.

### Failure Postconditions

- No thread is created.
- The form is redisplayed with validation errors.

## Business Rules

### BR-001: Authentication Required

Only authenticated users may start discussion threads.

### BR-002: Title Length

Thread titles must be between 5 and 200 characters.

### BR-003: Body Length

The opening post body must be between 10 and 10,000 characters.
